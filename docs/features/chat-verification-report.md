### 2. ✅ 工具执行状态修复

**问题**：
- 旧逻辑从 `msg.toolCalls` 提取工具
- 新消息格式使用 `parts` 数组

**修复**：
```typescript
// 修复后的提取逻辑
const toolExecutions = messages.flatMap((msg) => {
  if (msg.parts && msg.parts.length > 0) {
    return msg.parts
      .filter((part) => part.type === 'tool')
      .map((part) => ({
        id: part.id,
        name: part.name || '',
        input: part.input || {},
        result: part.result,
        isError: part.isError,
        startTime: part.startTime,
        endTime: part.endTime,
        durationMs: part.durationMs,
      }));
  }
  return msg.toolCalls || [];
});
```

**验证结果**：
- ✅ 测试通过
- ✅ 工具正确从 parts 数组提取
- ✅ 向后兼容旧格式（toolCalls）
