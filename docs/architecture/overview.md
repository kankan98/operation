# 系统架构概览

> **TL;DR**: 本系统采用前后端分离架构。前端 React 18 + TypeScript，后端 Express + SQLite + Drizzle ORM，通过 RESTful API 和 SSE 通信。支持多平台价格监控、智能警报和 AI 聊天助手。

## 数据流

### 用户操作流程

```
用户点击 → React 组件 → API 请求 → Express 路由
                ↑                          ↓
                │                    Service 层
                │                          ↓
        React Query 缓存              数据库操作
                ↑                          ↓
                └──────── 响应 ←───────────┘
```

## 技术栈总览

### 前端
- React 18 + TypeScript
- Vite (构建工具)
- Tailwind CSS v4 (样式)
- Zustand (全局状态)
- React Query (服务器状态)
- React Router (路由)

### 后端
- Node.js 18+ + Express
- TypeScript
- SQLite + Drizzle ORM
- Pino (日志)
- Better-SSE (流式响应)

### AI 服务
- Anthropic Claude (主)
- DeepSeek API (兼容 Anthropic 协议)
- OpenAI API (备选)
