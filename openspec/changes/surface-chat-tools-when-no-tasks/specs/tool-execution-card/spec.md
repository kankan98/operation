## ADDED Requirements

### Requirement: 工具执行记录在任务面板中自动暴露
系统SHALL在任务面板尚未被用户手动切换时，自动暴露新到达的工具执行记录，避免工具结果隐藏在空任务页签之后。

#### Scenario: 工具执行后到达且无任务
- **WHEN** 任务面板当前显示"任务概览"、当前会话没有任务、且工具执行记录从0变为至少1条
- **THEN** 任务面板自动切换到"工具执行"页签

#### Scenario: 用户已手动选择页签
- **WHEN** 用户已经点击过"任务概览"或"工具执行"页签
- **THEN** 后续任务或工具执行数据变化MUST NOT覆盖用户当前选择

#### Scenario: 响应式任务抽屉
- **WHEN** 用户在窄屏布局中打开任务抽屉且当前会话没有任务但存在工具执行记录
- **THEN** 抽屉内的任务面板默认选中"工具执行"页签并显示紧凑工具卡片列表
