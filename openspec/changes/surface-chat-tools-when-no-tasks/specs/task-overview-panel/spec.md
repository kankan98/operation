## ADDED Requirements

### Requirement: 任务面板默认显示可用内容
系统SHALL在右侧任务面板打开时优先显示当前会话已有的有效内容，避免在存在工具执行记录时默认展示空任务状态。

#### Scenario: 无任务但有工具执行
- **WHEN** 当前会话没有任务且存在至少一个工具执行记录
- **THEN** 任务面板默认选中"工具执行"页签并显示紧凑工具卡片列表

#### Scenario: 有任务时保持任务优先
- **WHEN** 当前会话存在至少一个任务
- **THEN** 任务面板默认选中"任务概览"页签

#### Scenario: 无任务且无工具执行
- **WHEN** 当前会话没有任务且没有工具执行记录
- **THEN** 任务面板默认选中"任务概览"页签并显示"暂无任务"空状态

#### Scenario: 会话切换重新选择默认页签
- **WHEN** 用户从一个会话切换到另一个会话
- **THEN** 任务面板MUST根据新会话的任务和工具执行记录重新选择默认页签
