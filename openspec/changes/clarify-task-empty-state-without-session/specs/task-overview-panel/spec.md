## ADDED Requirements

### Requirement: 任务空状态区分无会话语义
系统SHALL根据当前是否存在会话，显示语义准确的任务概览空状态说明。

#### Scenario: 无当前会话时提示先创建会话
- **WHEN** 用户在没有当前会话的聊天页打开任务概览且没有任务
- **THEN** 系统MUST显示“暂无任务”
- **AND** 系统MUST显示“发送第一条消息后，相关任务会在这里显示”
- **AND** 系统MUST NOT显示“当前会话还没有创建任务”

#### Scenario: 当前会话没有任务时保留会话语义
- **WHEN** 用户在已有当前会话的聊天页打开任务概览且该会话没有任务
- **THEN** 系统MUST显示“暂无任务”
- **AND** 系统MUST显示“当前会话还没有创建任务”
