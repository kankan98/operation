### 1. SSE 心跳机制
**优先级**: Medium
**建议**: 考虑在长时间工具执行期间添加定时心跳

**当前实现**:
```typescript
// 仅发送初始心跳
res.write(':ok\n\n');
```

**建议实现**:
```typescript
const heartbeatInterval = setInterval(() => {
  if (!res.writableEnded) {
    res.write(':ping\n\n');
  }
}, 30000); // 30秒心跳

// 在 finally 块中清理
finally {
  clearInterval(heartbeatInterval);
  res.end();
}
```
