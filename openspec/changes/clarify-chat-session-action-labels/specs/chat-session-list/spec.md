## ADDED Requirements

### Requirement: 会话操作菜单可访问名称区分
系统SHALL为会话操作菜单按钮提供与主新建会话按钮可区分的可访问名称。

#### Scenario: 未命名会话菜单不匹配新建会话按钮
- **WHEN** 会话列表同时显示主“新对话”按钮和无标题会话
- **THEN** 无标题会话的操作菜单按钮可访问名称MUST为“会话操作菜单：未命名对话”

#### Scenario: 已命名会话菜单包含菜单语义
- **WHEN** 会话列表显示有标题的会话
- **THEN** 该会话操作菜单按钮可访问名称MUST为“会话操作菜单：<会话标题>”

#### Scenario: 主新建会话按钮保持独立名称
- **WHEN** 用户或自动化按可访问名称“新对话”查找按钮
- **THEN** 系统MUST只匹配主新建会话按钮
